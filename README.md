# PyWiseClicker

[![License: GPL v2](https://img.shields.io/badge/License-GPLv2-blue.svg)](https://www.gnu.org/licenses/old-licenses/gpl-2.0.en.html)

## 核心功能
✅ **自定义点击参数**：可调节基础点击间隔（0.01~1.0秒）和随机误差（0~0.5秒），模拟真人点击；
✅ **灵活触发方式**：按住鼠标侧键（X1/X2）或中键启动连点，松开即停止；
✅ **紧急终止**：全局 `F10` 快捷键强制停止连点，杜绝卡死风险；

#基础操作
启动程序后，拖动滑块设置：
基础点击间隔（值越小点击越快，最小 0.01 秒）；
随机误差（模拟真人操作，建议设 0.1~0.3 秒）；
点击「▶ 激活」按钮，状态栏显示「已激活 - 按住侧键 / 中键开始连点」即生效；
按住鼠标侧键或中键，程序自动执行连点；松开按键，连点停止；
点击「⏸ 停用」可关闭连点功能，或按 F10 紧急终止。
